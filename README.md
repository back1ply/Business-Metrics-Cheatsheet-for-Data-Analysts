# Business Metrics Cheatsheet for Data Analysts

A searchable, filterable reference of business metrics across industries. Built as a Notion-style database viewer.

## Features

- **Table & Card Views** - Switch between layouts
- **Group By** - Group metrics by Industry, Category, or Stakeholder
- **Search** - Filter across all fields
- **Sortable Columns** - Click headers to sort
- **Collapsible Groups** - Expand/collapse sections
- **Detail Modal** - Click any metric for full details
- **Works Offline** - Drag-drop CSV for local use

## Usage

**Online:** Visit [GitHub Pages](https://back1ply.github.io/Business-Metrics-Cheatsheet-for-Data-Analysts/)

**Local:**
1. Clone the repo
2. Open `index.html`
3. Drag & drop `data/BMC_DB.csv` when prompted

## Industries Covered

- Technology / SaaS
- E-Commerce / Retail
- Healthcare / Pharmaceuticals
- Media / Entertainment / Streaming
- Manufacturing
- Telecommunications
- Insurance
- Energy / Utilities
- Logistics / Supply Chain
